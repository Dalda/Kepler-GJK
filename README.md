Kepler-GJK
==========

Autor: Dalimil Hájek
Maturitní projekt z informatiky.

Android aplikace pro Gymnázium Jana Keplera. 
Cílem je aplikace, která umí uživateli ukazovat aktuální změny v jeho rozvrhu a týdenní jídelníček školní jídelny.
